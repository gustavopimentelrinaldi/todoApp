* MongoDB
* NodeJS
* React

# How to run
```
git clone git@github.com:gustavopimentelrinaldi/todoApp.git
```
```
npm i -g npm
npm i
```
```
sudo mongod (keep running)
```
```
command -> npm run dev
```
Access localhost:3003/api/todos
